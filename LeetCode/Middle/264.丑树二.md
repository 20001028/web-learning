# 264.丑树II

**题目描述**

给你一个整数 `n` ，请你找出并返回第 `n` 个 **丑数** 。

**丑数** 就是只包含质因数 `2`、`3` 和/或 `5` 的正整数。

**题解**

- 动态规划

  ```javascript
  var nthUglyNumber = function(n) {
      let dp=new Array(n);
      let p2=0,p3=0,p5=0;
      dp[0]=1;
      for(let i=1;i<n;i++){
          dp[i]=Math.min(dp[p2]*2,dp[p3]*3,dp[p5]*5);
          if(dp[i]===dp[p2]*2)
              p2++;
          if(dp[i]===dp[p3]*3)
              p3++;
          if(dp[i]===dp[p5]*5)
              p5++;
      }
      return dp[n-1];
  };
  ```
